3.0 New Application

3.1 New Application

3.1.1: Adding new Academic Year for New Application

You can only add new application when the application process is open for an academic year. The Learning Advisor activates the application process when a new scholarship season starts.

 

Steps:

  1. Login to the scholarship tracking application.
  2. Select Support Data à Academic Year from the menu
  3. List of Academic Years page in the system opens
  4. Click on add button to start new academic year or click the edit button to edit status of academic year.

Academic year can have three status:

ADMISSION: New applicants can be accepted when the academic year is in this state

GRADE RECORDING: Grades of students can be recorded for that academic year

IN ACTIVE: Records under this academic year can not be edited.

 

3.1.2: New Application

Programme area/unit officers can input new application into the system for the programme area/unit they are responsible to. It is possible to give a staff permission to add new application to any programme area/unit.

 

Steps:

1.    Login to the scholarship tracking application using your user name and password.

2.    To open the new application form, click on ‘Student Applicant’  à ‘new applicant’ from the menu.

3.    The new application form opens.

4.     Enter the details of the new applicant. Red ‘*’ next to the label of an input indicate it is a required field and data has to be entered before submission.

 

If the primary school or JSS school of the student is not among the dropdown list, then click on the link ‘others’ to add it to the list of primary school.

Figure 23: Part of new applicant form. Click on the other link to add new JSS schools or primary school to the database.

When a new application is added to the record, it is given application points based on  parent/guardian occupation, aggregate grade, school and community. The point assigned can be  viewed by opening the application summary.

 

 

3.2 Download Application Summary

Application Summary is a word/pdf document generated from the application data entered, and it summarises details of the applicant. It is designed to aid the scholarship selection committee work. It contains information about the applicant with application points.

 

Steps:

1.    Login with your user name and password

2.    Select from the menu ‘Student Applicant’à ‘application summary’

3.    Select programme area/unit to generate summary for.

4.    Click generate button

5.    The summary document starts downloading.

6.    Each applicant is summarised in one page

 

3.3 Awarding Scholarship

After the selection committee identifies applicants to grant scholarship, the next step is to record the award against the selected applicants. The award is recorded with grant source and amount. Recording a scholarship award through the steps below does not mean the applicant is confirmed for sponsorship. When the student accepts the offer, then the award is confirmed.

 

Steps:

1.    Login using your user name and password.

2.    Select ‘Student Applicant’à ‘award scholarship’ from the menu.

3.    If you are responsible for one programme area/unit, then applicant from this programme area will be listed. If you are responsible for multiple programme area/unit, then you have to select the programme to list applicants

4.    Select the grant source to award scholarship from

5.    Click the award link next to the applicant to award scholarship.

6.    To cancel an award, click the cancel link

 

 

3.4 Generate Scholarship letter

Scholarship letter is a letter generated by the system to notify an applicant about the scholarship awarded. The body and the signatory of the letter can be edited before generating the letter. The letter includes the name of student, address, grant size and other information. The letter is generated as a word document.

 

Steps:

1.    Login with your username and password

2.    Select ‘Student Applicant’ à ‘scholarship letter’ from the menu.

3.    Enter the name of the signatory and title.

4.    If you are responsible for multiple program area/unit, then select the programme area/unit the scholarship letter will be generated to.

5.    Click on generate letter.

6.    The letter starts downloading.

Figure 24: Scholarship letter form allows user to generate letter to students informing them about the scholarship award.

3.6 Confirm Scholarship Award

Once the student accepts the scholarship offer, the next step is to confirm the student as a scholarship recipient in the system. The school the student is admitted to can be recorded during application, but it has to be confirmed at this point. The student has to confirm the school, the attendance type(boarder, day), the major (business, art, science), etc.

 

Steps:

1.    Login with your username and password.

2.    Select ‘Student Applicant’à’confirm application’.

3.    If you are responsible for multiple programme area/unit, select one from drop down list and click on ‘get list to confirm’ button.

4.    Click on the ‘confirm’ link next to the student to confirm.

5.    Enter the details like school, major, attendance, expiry year for the scholarship into the form on the right.

6.    Click ok to save the confirmation.
